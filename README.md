# RG13-pod-je-lava
pomozi pingvinu da nauci da leti

skaci po trampolinama

nemoj upasti u lavu!!!

## Potrebne biblioteke

- GLUT

## Instalacija i pokretanje

```bash
  git clone https://github.com/MATF-RG19/RG13-pod-je-lava.git
  cd RG13-pod-je-lava
  make
  ./lava
```
